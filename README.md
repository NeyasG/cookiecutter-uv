# Cookiecutter-uv Template

A lightweight template for python based projects using uv as a package manager.

## Features

- [uv](https://docs.astral.sh/uv/) for dependency management
- Supports src and flat layouts
- [ty](https://docs.astral.sh/ty/) and [ruff](https://docs.astral.sh/ruff/) for code quality
- [commitizen](https://commitizen-tools.github.io/commitizen/) for release management and commit discipline
- [pytest](https://docs.pytest.org/en/stable/) for testing
- [zensical](https://github.com/zensical/zensical) for generating documentation
- CI/CD with [GitHub Actions](https://github.com/features/actions)
- Pre-commit hooks with [pre-commit](https://pre-commit.com/)
